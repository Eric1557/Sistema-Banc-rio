# 🏦 Sistema Bancário em Java

Este projeto é uma aplicação simples de **Sistema Bancário** desenvolvida em **Java**, com foco em prática de **Programação Orientada a Objetos (POO)**.  
O sistema permite criar contas, realizar depósitos, saques e consultar saldo de forma interativa via console.

---

## 🚀 Funcionalidades
- Criar uma nova conta bancária
- Realizar depósitos
- Efetuar saques (com validação de saldo)
- Consultar saldo da conta
- Menu interativo em loop até o usuário decidir sair

---

## 📂 Estrutura do Projeto
- `AppBancario.java` → Classe principal com o menu e interação com o usuário
- `Banco.java` → Gerencia as contas bancárias (adicionar, buscar, etc.)
- `ContaBancaria.java` → Representa uma conta com número, titular e saldo

---

## 🛠️ Tecnologias Utilizadas
- **Java 17+** (funciona também em versões anteriores)
- **Scanner** para entrada de dados via console
- Estruturas de controle (`switch`, `while`, `try/catch`) para fluxo do programa
